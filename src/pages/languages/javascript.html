J a v a s c r i p t

Javascript(JS) es un lenguaje de programación que te permite crear contenido nuevo y dinámico, controlar archivos de multimedia, crear imágenes animadas...

JavaScript fue desarrollado originalmente por Brendan Eich de Netscape con el nombre de Mocha, el cual fue renombrado posteriormente a LiveScript, para finalmente quedar como JavaScript.

Javascript es un lenguaje que puede ser utilizado por profesionales y para quienes se inician en el desarrollo y diseño de sitios web. No requiere de compilación, ya que el lenguaje funciona del lado del cliente, los navegadores son los encargados de interpretar estos códigos.
Javascript es un lenguaje con muchas posibilidades, utilizado para crear pequeños programas que luego son insertados en una página web y en programas más grandes, orientados a objetos mucho más complejos. Con Javascript podemos crear diferentes efectos e interactuar con nuestros usuarios.

El código javascript podemos encontrarlo dentro de las etiquetas <body></body> de nuestras páginas web. Por lo general se insertan entre: <script></script>. También pueden estar ubicados en ficheros externos usando:
<script type="text/javascript" src="url/to/javascript/file.js"></script>

Ejemplo:

HTML:
<div class="example-container">
    <h1>Hola
        <span id="user-name">
            desconocido
        </span>
    </h1>
    <label for="name">
        First name:
    </label>
    <input id="name" name="name">
    <br>
    <button>
        ACTION
    </button>
</div>


CSS:
.example-container button {
  margin-top: 10px;

  /* Green */
  background-color: #4CAF50;

  border: none;
  color: white;
  padding: 16px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
  outline-color: unset;
  cursor: pointer;
}

.example-container button:active {
    background:#4e8450
}

.example-container label {
  color: blue;
  font-size: 14px;
}


JavaScrip:
let setName = function() {
    //Get the name written
    //in the input value.
    const n = document.getElementById('name');
    var name = n.value;
    name = name||"desconocido";

    //Set the name in the title.
    const u = document.getElementById('user-name');
    u.textContent = name;
};
