Angular es un marco de diseño de aplicaciones y una plataforma de desarrollo para crear aplicaciones eficientes y sofisticadas de una sola página. Angular es un framework para aplicaciones web desarrollado en TypeScript, de código abierto. Google es la empresa encargada del mantenimiento de Angular

En Angular los componentes son los elementos que definen la información o contenido que queremos mostrar en un pedazo de una pantalla. En los componentes también se define cómo se debe responder a distintos eventos

Uno de los propósitos de Angular es la separación completa entre el front-end y el back-end en una aplicación web.
