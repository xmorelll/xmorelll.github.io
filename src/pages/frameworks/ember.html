Ember

Ember.js es un marco de JavaScript productivo y probado en batalla para crear aplicaciones web modernas. Incluye todo lo que necesita para crear interfaces de usuario completas que funcionen en cualquier dispositivo. Ember está basado en el patrón Modelo-vista-modelo.

Permite a los desarrolladores crear aplicaciones web escalables de una sola página incorporando modismos comunes y mejores prácticas en el marco
